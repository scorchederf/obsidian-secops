# Atomic Red Team

<!-- generated-source-description-start -->
Atomic Red Team is a library of small, focused tests for emulating ATT&CK techniques in controlled environments. This vault imports Atomic YAML tests, keeps command and dependency blocks with executor-aware code fences, and maps each test back to its ATT&CK technique.

Use Atomic pages as validation references for testing whether detections, analytics, and telemetry collection work as expected. ATT&CK technique pages link to matching Atomic tests, and Sigma simulation metadata links to Atomic tests when GUIDs match.
